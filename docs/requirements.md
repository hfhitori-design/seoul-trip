# 开发需求

## 项目概述

首尔 Solo Trip — 个人首尔七日行程规划器，纯静态单页应用。

## 功能需求

### 行程管理
- [x] 按天分组展示行程（Day 1 ~ Day 7）
- [x] 每条行程包含：时间、地点、交通方式、备注
- [x] 新增行程（底部弹窗表单）
- [x] 编辑行程（点击编辑按钮 → 弹窗回填数据）
- [x] 删除行程（确认弹窗后删除）

### 天管理
- [x] 默认 7 天行程
- [x] 天标题可自定义编辑
- [x] 可新增额外天数
- [x] 可删除整天（含该天所有行程）

### 数据持久化
- [x] 行程数据存储在 localStorage
- [x] 首次使用加载默认示例数据
- [x] 数据版本自动升级
- [x] 支持一键重置回默认数据

### 交互体验
- [x] 行程卡片默认收起，点击标题展开/折叠
- [x] 编辑模式下所有卡片自动展开
- [x] 编辑按钮切换"编辑/完成"状态
- [x] 空天显示"暂无行程"提示
- [x] 弹窗从底部滑入（移动端友好）
- [x] 点击遮罩关闭弹窗

### 图片附件
- [x] 每个地点可添加多张图片
- [x] 缩略图显示在地点右侧（60×60）
- [x] 编辑模式下添加/删除图片
- [x] 全屏灯箱查看（左右切换、键盘导航）
- [x] IndexedDB 持久化图片

### Naver Map 跳转
- [x] 地点名淡蓝色可点击链接
- [x] App 优先 + 网页兜底

### 花销记录
- [x] 按地点记账 + 按天记账（零散花销）
- [x] 分类标签：餐饮/交通/购物/门票/其他
- [x] 日合计 + 全程总计自动计算
- [x] 编辑模式下内联表单快速录入

### 移动端适配
- [x] viewport 禁止缩放
- [x] 最大宽度 480px 居中
- [x] 触摸反馈（tap highlight 透明）

### 应急信息
- [x] 底部显示民宿地址
- [x] 底部显示地铁运营时间
- [x] 底部显示应急电话（112 / 1330）

## 待规划功能

- [ ] 行程数据导出/导入
- [ ] 打印友好样式
- [ ] 行程之间拖拽排序
- [x] Service Worker 离线缓存
