# 设计规范

## 设计原则

- **移动端优先**：以手机屏幕为主要目标设备
- **极简克制**：低饱和度配色，大量留白
- **操作明确**：编辑模式切换，关键操作有确认

## 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#f8f7f4` | 暖灰白 |
| 卡片背景 | `#fff` | 纯白 |
| 主文字 | `#2c2c2c` | 近黑 |
| 次要文字 | `#666` | 中灰 |
| 辅助文字 | `#888` / `#999` | 浅灰 |
| 占位/禁用 | `#aaa` / `#bbb` | 更浅灰 |
| 主操作按钮 | `#2c2c2c` | 近黑底白字 |
| 必填标记 | `#e55` | 红色 |
| 危险按钮 | `#fff0f0` | 浅红底 |
| 边框/分割线 | `#f0efeb` | 暖灰 |
| 虚线按钮 | `#ddd` | 浅灰 |

## 弹窗

- 位置：底部弹出（`align-items: flex-end`）
- 最大高度：85vh，超出滚动
- 动画：`slideUp` 0.25s ease（translateY 100% → 0）
- 遮罩：`rgba(0,0,0,0.4)`
- 宽度：100%，最大 480px

## 布局

- 最大宽度：480px，水平居中
- 卡片默认折叠（`display: none`），点击展开（`display: block`）
- 编辑模式下所有卡片强制展开
